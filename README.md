# 佛山順德容桂 3 日遊 · 投票站

6/30 - 7/2 出發前,9 個朋友一齊投票決定食邊間餐廳、去邊個景點、唱邊間 KTV 嘅單檔 HTML 投票站。

**而家共 30 個候選地點**:17 餐廳 + 5 景點 + 3 夜店/KTV + 5 養生/SPA

## 用法


### 0. 住宿
頁面頂部會顯示住宿酒店 chip · 撳開就到 trip.com:
- **佛山柏希酒店** — 順德區容桂東堤路16號
- 連結: https://www.trip.com/w/DecTOJYQ5V2

改酒店資料:打開 `index.html`,搵 `<script type="application/json" id="users-data">`,改 `hotel` 個 field(name / address / url)。

### 0.5 大眾點評連結
每間餐廳卡片右上角會顯示「↗ 點評」chip · 撳開直接跳去大眾點評 shop 頁(新分頁):
- 17 餐廳入面有 16 間用 `dianping.com/shop/{id}` 直接連去店家頁
- `百搭粥神` 用 `dianping.com/search/keyword/...` 因為佢冇官方店頁

改連結:打開 `index.html`,搵 `<script type="application/json" id="places-data">`,每個 restaurant 個 entry 加 `"dianping": "https://..."` 即可。

### 0.7 介面(更新版)
撳入即見 9 個大掣(揀你個名) → 進去見 4 個 tab(`想食/想玩/想唱/想hea`)+ 全部,卡片底部全闊 `投畀呢度` 掣 → 1 撳即投。
主揪 Match 會喺 `⋯ 選單` 見到額外 `主揪總覽` 同 `全部地點地圖`。

- 重複 import 9 個人嘅 JSON
- 去「主揪總覽」tab 睇全部結果

## 技術

- 單檔 HTML,無 build,無後端
- 資料 hardcode(25 個地點)
- 投票 / 留言存 `localStorage`(每個裝置獨立)
- Noto Serif TC + Noto Sans TC 字體(Google Fonts)
- 真實 Unsplash 食物相

## 注意嘢

- **每個裝置獨立投票**:資料喺 localStorage,唔會自動 sync。要睇其他人嘅票,必須匯入 JSON
- **搵唔到嘅名會累計失敗次數**,3 次後鎖 30 秒(防暴力)
- **轉用戶唔會清資料**:投票 / 留言會留住
- **唔支援密碼重設**:加新成員,主揪喺 `users` 陣列加名就得
